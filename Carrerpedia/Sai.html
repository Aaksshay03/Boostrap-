Inline Elements

a - Anchor Used for creating hyperlinks. abbr - Abbreviation Represents an abbreviation or acronym and can include a title attribute for additional information. b - Bold Represents bold text cite - Citation Used to reference the title of a creative work code - Code Represents a fragment of computer code em - Emphasis Represents emphasized text, typically displayed in italics i - Italic Represents italicized text img - Image Used to embed images in a web page span - Span Generic container that does not carry any semantic meaning and is often used for styling or scripting purposes strong - Strong: Represents strongly emphasized text, typically displayed in bold br - Line Break Represents a line break within text sup - Superscript Represents superscript text sub - Subscript Represents subscript text u - Underline Represents underlined text mark - Marked/Highlighted Text Represents text that has been marked or highlighted.
